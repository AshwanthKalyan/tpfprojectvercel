Product Requirements Document (PRD)
1. Product Overview
Product Name

NITT Project Hub (working title)

Product Description

NITT Project Hub is a web platform that allows students of National Institute of Technology Tiruchirappalli to post project ideas or ongoing projects and recruit collaborators from within the college.

Students can:

Post projects they want to build

Discover projects posted by other students

Express interest in joining a project

Upload their resume

Allow project creators to review applicants and accept collaborators

The goal is to encourage collaboration, interdisciplinary work, and faster team formation for academic, research, and hackathon projects.

2. Problem Statement

Students often have:

Good project ideas but no team

Skills but no project to join

Difficulty finding collaborators across departments

Current methods:

WhatsApp groups

Discord servers

Personal connections

These are unstructured and inefficient.

A centralized system will make project discovery and team formation easier.

3. Goals & Objectives
Primary Goals

Enable students to post projects easily

Allow students to find projects matching their interests

Provide a way to apply to projects

Allow project creators to review applicants and select collaborators

Success Metrics

Number of projects posted

Number of project applications

Number of teams successfully formed

Active student users

4. Target Users
Primary Users

Students from National Institute of Technology Tiruchirappalli

User Types
1. Project Creator

A student who posts a project and wants collaborators.

Actions:

Create project listing

View interested applicants

Review resumes

Accept / reject applicants

2. Applicant

A student who wants to join projects.

Actions:

Browse projects

Apply to projects

Upload resume

Track application status
